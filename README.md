# 📰 Dev Times

**Dev Times** is a personal newspaper style developer portfolio built with React. It presents my background, experience, featured projects and professional journey through an editorial inspired layout.
</div>

## Tech Stack

<div align="center">

[![My Skills](https://skillicons.dev/icons?i=react,tailwindcss)](https://skillicons.dev)

</div>

- **React** - UI components and app structure
- **Tailwind CSS** - Modern styling and responsive design


## Highlights

- Newspaper-inspired visual design
- Hero section with personal introduction
- Timeline of experience and growth
- Featured project cards
- Responsive layout for desktop and mobile

## Project Structure

- `src/App.jsx` - Main portfolio layout
- `src/Components/` - Navbar, hero, timeline, projects, footer and aside sections
- `public/` - Images and static assets
